## Installation

___

1. Clone the repo
```
git clone https://github.com/anantl911/Anant-Space-portfolio.git
```
2. Install dependencies for frontend
```
npm i
```

You will now be able to access the project at:
> localhost:5173 
>  assuming you kept the port remains unchanged.
